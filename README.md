웹툰 다운로드는 직접 해당 사이트에서 크롤링을 하지 않습니다.

제 서버에서 크롤링후 텔레그램으로 전송한 암호화코드를 전송받아 사용자의 DB에 저장하여 다운로드 하는 방식입니다.

이유는 요청을 많이 하게 되면 차단이 될수가 있고 또 코드도 매번 바꿔야 하는 번거로움 때문입니다.
