1. 수신용 봇 준비
현재 시스템은 이중 해독(Double Decode) 기술을 사용하여 데이터를 수집합니다. 서버가 데이터를 쏘는 채널에 접근할 수 있는 본인의 봇 토큰이 필요합니다.

2. 봇 토큰 입력
상단 '내 봇 토큰' 칸에 텔레그램 @BotFather에서 발급받은 HTTP API 토큰을 입력하세요.

3. 채널 연결 확인
데이터가 전송되는 채널(예: @webtoon_db)에 본인의 봇이 관리자(Admin)로 추가되어 있어야 메시지를 읽어올 수 있습니다.
